# Pico-TG-Tamagotchi-Emulator

## Hardware
- Raspberry Pi Pico
- SSD1306 OLED (I2C)
- 3 Push Buttons
- Battery via VSYS

## Features
- FOOD / WASH system
- Status decay
- Jump mini-game
- Battery standalone boot (main.py auto run)

## Pin Mapping
OLED SDA → GP20  
OLED SCL → GP21  
Button A → GP16  
Button Left → GP17  
Button Right → GP18  

## How to Run
Save files to Pico as main.py and power via VSYS.
